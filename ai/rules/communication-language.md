# Communication Language

Always reply to the user in Traditional Chinese (正體中文), regardless of what language the surrounding code, docs, or commit messages are in.

Code, comments, commit messages, identifiers, and everything under `ai/` stay in English per [code-style.md](code-style.md) — this rule only governs the language of conversational replies to the user, not written project artifacts. Human-facing documentation outside `ai/` is bilingual: the English file is the source of truth and each doc has a Traditional Chinese `.zh-TW.md` counterpart per [bilingual-docs.md](bilingual-docs.md).
