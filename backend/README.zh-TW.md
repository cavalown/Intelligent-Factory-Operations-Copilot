## 說明

[Nest](https://github.com/nestjs/nest) 框架 TypeScript 起始 repository。

## 專案設定

```bash
$ npm install
```

## 編譯與執行專案

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 執行測試

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## 部署

當你準備好將 NestJS 應用程式部署到正式環境時，可以採取一些關鍵步驟，確保它以最有效率的方式運行。詳情請參閱[部署文件](https://docs.nestjs.com/deployment)。

如果你在尋找雲端平台來部署 NestJS 應用程式，可以看看 [Mau](https://mau.nestjs.com)——我們在 AWS 上部署 NestJS 應用程式的官方平台。Mau 讓部署變得簡單快速，只需幾個簡單步驟：

```bash
$ npm install -g @nestjs/mau
$ mau deploy
```

使用 Mau，只要點幾下就能完成部署，讓你專注在開發功能上，而不是管理基礎設施。

以下是一些使用 NestJS 時可能派上用場的資源：

- 使用 [NestJS Devtools](https://devtools.nestjs.com) 視覺化你的應用程式關係圖，並即時與 NestJS 應用程式互動。
- 專案需要協助（兼職到全職皆可）？請參考官方[企業支援](https://enterprise.nestjs.com)。
