# Vue 3 + TypeScript + Vite

這個範本可以協助你開始使用 Vue 3、TypeScript 與 Vite 進行開發。範本採用 Vue 3 的 `<script setup>` 單一檔案元件（SFC），詳情請參閱 [script setup 文件](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup)。

關於建議的專案設定與 IDE 支援，請參閱 [Vue 文件的 TypeScript 指南](https://vuejs.org/guide/typescript/overview.html#project-setup)。
